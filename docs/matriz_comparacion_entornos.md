# Matriz de comparación técnica de entornos

Completa esta matriz después de ejecutar el laboratorio. Usa una valoración de 1 a 5.

| Criterio | Pregunta guía | Valoración 1 a 5 | Justificación técnica | Evidencia usada |
|---|---|---:|---|---|
| Funcionalidad | ¿El entorno permitió abrir carpetas, editar código, usar terminal, ejecutar scripts, revisar errores y generar reportes? | 4 | El proyecto se abrió en VS Code y se ejecutó `03_generar_reporte.py`, generando el reporte técnico. | Archivo `reporte_tecnico_semana13.txt`, estructura del proyecto |
| Compatibilidad | ¿Funcionó correctamente con tu equipo, sistema operativo, Python, CSV y librerías? | 3 | Se ejecutó un script de Python en Windows y se procesaron CSV, pero no hay evidencia completa de instalación exitosa de todas las dependencias. | Terminal usada, archivo de reporte |
| Costo | ¿Es gratuito? ¿Requiere licencia, internet constante o equipo potente? | 5 | VS Code es gratuito y el proyecto usa Python local, por lo que no parece requerir software de pago. | Conocimiento del entorno, archivos locales |
| Rendimiento | ¿Procesó los CSV grandes sin bloquearse o demorarse demasiado? | 4 | El reporte se generó con datos de miles de registros sin bloqueo visible en la terminal. | Salida de `reporte_tecnico_semana13.txt` |
| Facilidad de uso | ¿Fue fácil encontrar archivos, terminal, errores, extensiones y resultados? | 4 | Identifiqué rápidamente las carpetas `data`, `src`, `reportes`, `docs` y el terminal integrado. | Estructura del proyecto |
| Documentación | ¿La guía, comentarios del código y mensajes fueron suficientes para avanzar? | 4 | Hay documentación en `docs` y el proyecto incluye instrucciones claras de los pasos básicos. | README.md, `docs`, `reporte_tecnico_semana13.txt` |
| Recomendación final | ¿Recomendarías este entorno para una institución educativa rural? | 4 | Sí, porque permite trabajar datos rurales con herramientas gratuitas y locales, aunque conviene verificar dependencias antes. | Conclusión del informe, evidencia de ejecución |

## Interpretación sugerida

- 1: Muy bajo o no cumple.
- 2: Cumple parcialmente con dificultades importantes.
- 3: Cumple de manera aceptable.
- 4: Cumple bien y con pocas dificultades.
- 5: Cumple de forma excelente para el contexto.
